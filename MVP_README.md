# TimeAgree - MVP

## 🎉 Что реализовано (MVP)

### ✅ Готовые функции:

1. **Управление календарём**
   - Интерактивный календарь с навигацией по месяцам
   - Визуализация занятых дней (красным цветом)
   - Просмотр занятости на выбранную дату

2. **Добавление занятого времени**
   - Одноразовые слоты (выбор даты и времени)
   - Повторяющиеся слоты (выбор дней недели)
   - Описание для каждого слота
   - Удаление слотов

3. **Поиск свободного времени**
   - Алгоритм поиска пересечений свободного времени
   - Отображение найденных слотов с длительностью
   - Поиск на ближайшие 7 дней
   - Минимальная длительность 30 минут

4. **Настройки**
   - Базовый экран настроек
   - Кнопка очистки данных

5. **База данных**
   - Prisma ORM с SQLite
   - Модели User и Slot
   - API endpoints для CRUD операций

## 🚀 Как использовать

### Запуск приложения:

Приложение уже запущено автоматически. Просто откройте Preview Panel справа.

### Основные функции:

#### 1. Добавить занятое время:

1. Откройте вкладку "Моё время"
2. Выберите дату в календаре
3. Нажмите кнопку "Добавить"
4. Заполните форму:
   - Описание (например: Работа, Встреча)
   - Тип: Одноразово или Каждую неделю
   - Для одноразового: выберите дату
   - Для повторяющегося: выберите дни недели
   - Укажите время начала и конца
5. Нажмите "Сохранить"

#### 2. Просмотреть занятость:

- В календаре занятые дни отмечены красным
- Нажмите на дату, чтобы увидеть все слоты на этот день
- Нажмите "Удалить" чтобы удалить слот

#### 3. Найти свободное время:

1. Откройте вкладку "Найти время"
2. Нажмите "Найти свободное время"
3. Система покажет все свободные слоты минимум на 30 минут

## 💡 Как тестировать

### Тест-кейс 1: Добавить одноразовый слот

1. Откройте вкладку "Моё время"
2. Нажмите "Добавить"
3. Выберите тип "Одноразово"
4. Выберите завтрашнюю дату
5. Установите время 10:00 - 12:00
6. Нажмите "Сохранить"
7. Проверьте, что слот появился в списке

### Тест-кейс 2: Добавить повторяющийся слот

1. Откройте вкладку "Моё время"
2. Нажмите "Добавить"
3. Выберите тип "Каждую неделю"
4. Выберите Пн, Ср, Пт
5. Установите время 09:00 - 18:00
6. Нажмите "Сохранить"
7. Проверьте, что эти дни отмечены красным в календаре

### Тест-кейс 3: Найти свободное время

1. Добавьте несколько занятых слотов
2. Откройте вкладку "Найти время"
3. Нажмите "Найти свободное время"
4. Проверьте, что система показывает свободные слоты
5. Слот должен быть минимум 30 минут

## 🐛 Известные ограничения (MVP)

- Работает только с одним демо-пользователем
- Нет поддержки групп (будет в следующей версии)
- Нет Telegram интеграции (будет в следующей версии)
- Данные хранятся локально (SQLite)
- Нет синхронизации между устройствами
