## 🎯 Основные изменения дизайна

### 1. Тёмная тема (Dark Mode)
```
Текущий: Светлый фон, много белого
Новый: Тёмный фон (#0a0a0a), экономит батарею
```

### 2. Минимализм
```
Текущий: Перегруженный интерфейс, много элементов
Новый: Чистый, air spaces, фокус на контенте
```

### 3. Цветовая палитра
```
Текущий: Много цветов (синий, зелёный, оранжевый, фиолетовый, серый)
Новый: Индиго/фиолетовые акценты (#6366f1) + нейтральные
```

### 4. Градиенты
```
Текущий: Жёсткие градиенты
Новый: Мягкие, профессиональные градиенты
```

### 5. Иконки и эмодзи
```
Текущий: Иконки Lucide React
Новый: Эмодзи (визуально понятнее, меньше кода)
```

## 🎨 Дизайн-токены

### Цвета
```css
--bg-primary: #0a0a0a
--bg-secondary: #111111
--bg-tertiary: #1a1a1a
--bg-card: #161616

--text-primary: #ffffff
--text-secondary: #a0a0a0
--text-muted: #666666

--accent: #6366f1
--accent-hover: #818cf8
```

### Border Radius
```css
--radius-sm: 8px   (кнопки, инпуты)
--radius-md: 12px  (карточки)
--radius-lg: 16px  (модальные окна)
```

### Тени
```css
Минимальные тени, больше на границах
```

### Анимации
```css
Duration: 150-200ms
Easing: ease-out
Свойства: transform, background, border
```

## 📱 Компоненты

### 1. Header
- Sticky + backdrop-blur
- Градиентный логотип
- Полупрозрачный

### 2. Sidebar
- Sticky навигация
- Hover: background shift
- Active: accent color

### 3. Calendar
- 7x7 grid
- Square cells (aspect-ratio: 1:1)
- Hover: scale(1.05) + border highlight
- Selected: accent background
- Has slots: dot indicator

### 4. Quick Templates
- 4-column grid (mobile: 2, mobile-sm: 1)
- Hover: translateY(-2px)
- Иконка + название + время

### 5. Time Slot Cards
- Flex layout
- Icon (44x44px) + content + actions
- Hover: shadow + border highlight
- Drag & drop ready

### 6. Statistics
- Progress bars with gradients
- Animation on load
- Clean typography
