## Технические проблемы и решения

### Проблема 1: Prisma BigInt сериализация
**Ошибка**: `Do not know how to serialize a BigInt`
**Причина**: JSON.stringify не умеет сериализовать BigInt
**Решение**: Конвертация BigInt в строку через `typeof value === 'bigint' ? value.toString() : value`

### Проблема 2: Prisma schema кэширование
**Ошибка**: `Unknown argument 'memberCount'`
**Причина**: Prisma Client генерируется один раз и кэшируется, схема в базе обновилась но клиент нет
**Решение**: Переход на raw SQL во всех API endpoints

### Проблема 3: WebApp кнопки в группах
**Ошибка**: `Bad Request: BUTTON_TYPE_INVALID`
**Причина**: Telegram API не поддерживает web_app кнопки в групповых чатах
**Решение**: Убрать кнопки из групповых чатов, оставить только в личных

### Проблема 4: Общая переменная loading
**Ошибка**: Кнопка "Сохранить" постоянно заблокирована
**Причина**: Использование одной переменной для разных асинхронных операций
**Решение**: Разделить состояния на `savingSlot` и убрать общее `loading`
